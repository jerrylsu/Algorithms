
### dp状态：
```
dp[i] => max
```

### dp初始状态：
```
dp[0] = nums[0]
```

### dp状态转移方程：
```
dp[i] = max(dp[i - 1] + nums[i], nums[i])
```


```python
class Solution:
    def maxSubArray(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        if not nums: return
        dp, dp[0], res = [0, 0], nums[0], nums[0]
        for i in range(1, len(nums)):
            x, y = i % 2, (i - 1) % 2
            dp[x] = max(dp[y] + nums[i], nums[i])
            res = max(dp[x], res)
        return res
```


```python
s = Solution()
nums = [1,-2]
```


```python
s.maxSubArray(nums)
```
